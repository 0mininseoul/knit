## 🎥 **2. 시연 시나리오 및 기술 구현 (Fake Tech)**

### **Scene 1: AI 음성 온보딩 (30초)**
- **시나리오**: 노인이 앱을 켜면 AI가 말을 걸고, 대답하면 프로필이 자동 완성됨.
- **실제 구현 (Simulated)**:
    - **AI Voice**: 미리 녹음된 성우/TTS 오디오 파일 재생 (`<audio>` 태그).
    - **STT (Listening)**: 실제 음성 인식 안 함. 사용자가 아무 말이나 하면 마이크 파형 애니메이션(Web Audio API)이 반응하고, 1.5초 후 "인식 성공" 처리.
    - **프로필 생성**: 하드코딩된 데이터가 타이핑 효과로 폼에 채워짐.

### **Scene 2: AI 매칭 (30초)**
- **시나리오**: AI가 데이터를 분석하여 최적의 3인을 매칭해줌.
- **실제 구현 (Simulated)**:
    - **분석**: Lottie 애니메이션 (복잡한 그래프가 움직이는 연출).
    - **결과**: `setTimeout(3000)` 후 미리 준비된 "호환성 98%" 매칭 결과 화면 출력.
    - **백엔드 로직**: 없음 (프론트엔드에서 State 변경).

### **Scene 3: 그룹 콜 & 채팅 (30초)**
- **시나리오**: 버튼 한 번으로 안부를 전하고, 3명이 즐겁게 통화하며 채팅도 나눔.
- **실제 구현 (Simulated)**:
    - **One-touch**: 버튼 클릭 시 CSS 애니메이션 + 사운드 효과.
    - **Group Call UI**: 
        - 3명 프로필 사진 + 말할 때마다 움직이는 파형(Visualizer).
        - **Chat UI**: 하단에 실시간 채팅창 구현. 가짜 메시지가 타이밍에 맞춰 자동 스크롤되며 올라옴.
        - 실제 통화 기능(WebRTC) 및 채팅 백엔드 없음.
        - 영상 촬영 시에는 배우 3명이 실제로 다른 전화를 쓰거나 현장에서 연기.

### **Scene 4: AI 모더레이터 (30초)**
- **시나리오**: 오프라인 만남에서 스마트폰이 대화를 주도함.
- **실제 구현 (Simulated)**:
    - **화면**: 지도 UI + "대화 주제: 고향 이야기" 카드 표시.
    - **음성**: "자, 이제 어색함을 풀기 위해 고향 이야기를 해볼까요?" (녹음 파일 재생).
    - **인터랙션**: 화면 터치 시 다음 주제로 넘어가는 단순 트리거.

## 🛠 **3. 기술 스택 (Lightweight)**

### 💻 Frontend (핵심)
- **Framework**: **React** (Vite) + TypeScript
- **Styling**: **Tailwind CSS** + **Framer Motion** (고퀄리티 애니메이션 필수)
- **State Draft**: Zustand (간단한 전역 상태 관리)
- **Deployment**: **Vercel** (무료 티어)

### ☁️ Backend & Data (최소화)
- **DB**: **Supabase** (PostgreSQL) - *Free Tier*
    - 용도: 로그인한 척하는 사용자 프로필 정보, 가짜 채팅 데이터 저장.
- **API**: 없음 (대부분 Client-side에서 Mock Data 처리).
- **Auth**: 없음 (데모용 계정 `demo_user` 자동 로그인 처리).

### 🤖 AI & Voice (Fake)
- **Voice Gen**: ElevenLabs (사전 녹음용)
- **Images**: Midjourney/DALL-E 3 (가상 프로필 사진 생성)

## 🚨 **6. 주의사항**
1. **이것은 실제 제품이 아닙니다**: 투자 유치 후 폐기하고 새로 짜야 할 수도 있는 "일회용 코드"임을 인지해야 함.
2. **타이밍이 생명**: 영상 촬영 시 배우의 연기와 앱의 오디오 재생 타이밍이 딱 맞아야 함.
3. **디테일한 연출**: "가짜"지만 "진짜"처럼 보여야 하므로, 로딩 시간, 버튼 눌림 효과 등 마이크로 인터랙션에 공을 들여야 함.
